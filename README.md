### 玩Android随便玩玩项目

使用Kotlin编写，利用玩Android开发的API，开发的一个练手APP

#### 相关技术

- Kotlin
- MVVM
- DataBinding
- Retrofit
- Glide
- Room

#### 主要完成的功能

- 首页最新文章
- 项目
- 公众号
- 体系
- 网站导航
- 登录、注册
- 收藏
- 文章搜索
