### 玩Android随便玩玩项目

#### 相关技术

- Kotlin
- MVVM
- DataBinding
- Retrofit
- Glide

#### 功能描述

- 首页最新文章
- 项目
- 公众号
- 体系
- 网站导航
- 登录、注册
- 收藏
- 文章搜索
